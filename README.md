# Diabetes Analysis & Prediction Project

## Overview
A comprehensive diabetes data analysis project with interactive Streamlit dashboard.

## Project Structure
- `data/` - Dataset storage (raw, processed, reference)
- `notebooks/` - Jupyter notebooks for analysis
- `docs/` - Research papers and documentation
- `app.py` - Streamlit web application

## Setup Instructions
1. Clone the repository
2. Create virtual environment: `python -m venv venv`
3. Activate: `source venv/bin/activate` (Mac/Linux) or `venv\Scripts\activate` (Windows)
4. Install dependencies: `pip install -r requirements.txt`
5. Run app: `streamlit run app.py`
